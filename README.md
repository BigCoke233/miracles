# 特色
- 响应式/自适应设计
- Pjax 全站无刷新
- 夜间模式
- 前台配置面板
- 前台登录
- 代码高亮/行号
- 内置 Live2d
- 简洁大气的设计
- 灵活的友链功能
- 方便排版的短代码
- 文字可读性高
- 一个完善的 wiki（强调）

# 使用
1. 下载最新的 [Release](https://github.com/BigCoke233/miracles/releases)
2. 解压后，确认目录名为`Miracles`（M 大写）
3. 将文件夹放入`usr/themes`
4. 到后台启用主题
5. 配置主题
> 有问题请先查看 [Wiki](https://github.com/BigCoke233/miracles/wiki)

<details>
<summary><strong>使用开发版</strong></smmary>
